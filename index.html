ADMINISTRASI

Ketentuan Pendaftaran dan Pemberkasan

Pastikan semua berkas disiapkan dengan lengkap sebelum mendaftar.

  1. Setiap peserta wajib memenuhi seluruh persyaratan sesuai ketentuan.
  2. Setiap pangkalan mengisi dan mengirimkan lembar data peserta. (https://gqr.sh/t4ru)
  3. Setiap pangkalan mengirimkan scan/foto kartu identitas peserta (KTP/KTA/Kartu Pelajar) secara kolektif dalam satu file PDF dengan format file: Kartu Identitas_Nama Pangkalan.pdf (Contoh: Kartu Identitas_MAN 2 Kota Malang).
  4. Setiap pangkalan mengirimkan pas foto peserta ukuran 3x4 dengan latar belakang merah secara kolektif dalam satu file PDF dengan format file: Foto Peserta_Nama Pangkalan.pdf (Contoh: Foto Peserta_MAN 2 Kota Malang).
  5. Setiap pangkalan mengirimkan bukti telah mengikuti akun Instagram LAGA AKSI dan Prandutama (@lagaaksi_ dan @prandutama).
  6. Setiap pangkalan mengirimkan bukti pembayaran sebesar Rp20.000/peserta pada No. Rek. 973944985 BNI a.n Anastasya Putri secara kolektif dalam satu file PDF dengan format file: Pembayaran_Nama Pangkalan.pdf (Contoh: Pembayaran_MAN 2 Kota Malang).
  7. PENTING: Setiap pangkalan wajib membawa hard file berkas 3, 4, 5, dan 6 pada saat registrasi di dalam map kertas.
    📘 Map Biru — Untuk Semua Berkas Pangkalan
